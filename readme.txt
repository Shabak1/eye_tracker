Моторы подключаются через Arduino UNO и специальный драйвер

Схема подключения двух моторов через драйвер l298n по ссылке ниже
https://роботехника18.рф/wp-content/uploads/2019/02/motor-arduino-l298n.jpg

Скетч для Arduino находится в файле "motors.cpp". Там прописана логика управления моторами через платформу Arduino
В файле"code.py" находится код программы управления коляской

Для сохранения данных пользователя (пока только имя), необходимо создать файл с расширением .json и указать путь к нему в соответствующей строке
